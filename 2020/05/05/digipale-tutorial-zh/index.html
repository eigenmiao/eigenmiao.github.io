DigitalPalette 教程
2020-05-05|软件
字数总计: 5.4k|阅读时长: 16 分钟

DigitalPalette 是一款免费开源软件，它可以帮助您从色彩转轮或者已有图像中生成一组符合一定调和规则的色彩，并应用到您的创作中。您可以将心仪的色彩组储存在本软件中，并在下次需要时再提取出来。您也可以将创建好的色彩组或者色彩仓库导出为色彩文档，并放置在任意地方，包括网络存储。另外，您也可以将色彩文档分享给您的好友，或者导入其他软件中，如 Adobe Photoshop、GIMP、Krita、Pencil 2D 以及优动漫 Paint 等。DigitalPalette 使用 PyQt5 编写，可以在 Windows 和 Linux 等主流操作系统上正常运行。

安装

  1. 双击打开下载的安装程序，点击“下一步”。
  2. 阅读许可协议。若您同意则勾选“我接受许可协议”，点击“下一步”。
  3. 选择安装路径，点击“安装”开始安装。
  4. 点击“完成”结束安装，运行 DigitalPalette。

界面布局

DigitalPalette 的界面布局如下图所示，包括：

  • 标题栏（红色区域），包含软件名称（DigitalPalette）和当前版本（v2.2.8-开发版）。

  • 菜单栏（黄色区域），包含“文档”、“编辑”、“视图”、“窗口”以及“帮助”选项卡。

  • 状态栏（粉色区域），提供鼠标当前所指位置元素的提示信息。

  • 工作区域（绿色区域），从色彩转轮中创建一组色彩、从色彩图像中定位一组色彩以及将创建的色彩组添加到色彩仓库中。

  • 规则窗口（青色区域），设置色彩组的调和和同步规则。

  • 检测窗口（青色区域），对图像进行边缘和通道检测。

  • 操作窗口（蓝色区域），执行文档和视图操作。

  • 处理窗口（蓝色区域），对图像进行提取、滤波、快照、裁剪和缩放处理。

  • 模式窗口（蓝色区域），设置色彩组结果的显示模式。

  • 调整窗口（蓝色区域），对图像和色彩组进行位置、尺寸和色彩调整。

  • 结果窗口（紫色区域），查看和修改色彩组结果。

快速开始

示例 1：从色彩转轮中创建一组色彩

  1. 打开安装好的 DigitalPalette 软件。
  2. 在菜单栏中点击视图并在下拉列表中选择“转轮”，切换到转轮视图（默认视图）。在规则窗口调和选项框内选择“相似组”。然后拖动色彩转轮上的标签，选择一组合适的相似色彩。
  3. 在操作窗口文档按钮框内点击“导出”，将这组色彩导出为 Adobe 样板文档（\*.aco）。
  4. 打开 Adobe Photoshop 或者 GIMP 等图像编辑软件，导入上一步导出的文档。然后使用这个色板给图像着色。
  5. 返回 DigitalPalette，并在规则窗口调和选项框内选择“三色组”。然后拖动色彩转轮上的标签，选择一组合适的三色色彩。
  6. 用同样方法给图像着色，并将着色后的图像与上面的图像对比。

示例 2：从色彩图像中定位一组色彩

  1. 打开安装好的 DigitalPalette 软件。
  2. 准备好一张彩色的图片。示例里的图片由 Icyphantom 创建，采用知识共享署名-非商业性使用协议 3.0（CC BY-NC 3.0）发布在我的世界社区网站上。
  3. 在菜单栏中点击视图并在下拉列表中选择“图像”，切换到图像视图。
  4. 双击工作区域，打开上一步下载的图像。
  5. 在处理窗口提取按钮框内点击“明亮多彩”。图像上出现五个环形色彩标签，这些标签的颜色对应于结果窗口内的五个方块的颜色，也就是从图像中提取的一组色彩。若您对这组色彩不满意，您也可以像在色彩转轮里一样拖动这些标签，以获取特定位置的色彩。
  6. 使用从这张图像中得到的色彩来着色另一幅图像。

示例 3：将创建的色彩组添加到色彩仓库中

  1. 打开安装好的 DigitalPalette 软件。
  2. 从色彩转轮中创建一组色彩，或者从色彩图像中定位一组色彩。
  3. 在菜单栏中点击视图并在下拉列表中选择“仓库”，切换到仓库视图。
  4. 双击左上角的空白图标，将搭配好的色彩组添加到仓库中。
  5. 右键单击添加的色彩组图标，然后点击在下拉菜单中的“导入”、“导出”、“删除”、“详情”或“添加”来修改色彩组的信息。例如，选择“详情”，在弹出的对话框内可以修改色彩组的名称（DigiPale 色彩组）和描述（无），查看色彩组的调和规则（相似组）、创建时间（上）和修改时间（下）等。修改完毕后点击确定即可。

工作区域

工作区域是您搭配、修改、储存以及查看色彩组的主要区域。在这里您可以从色彩转轮中创建一组色彩，可以从色彩图像中定位一组色彩，也可以将创建的色彩组添加到色彩仓库中。

色彩转轮视图如下图所示，是工作区域的默认视图。若当前工作区域的视图不是色彩转轮，您可以通过单击菜单栏中的“视图”，然后选择“转轮”（或者点击操作窗口中的“创建”）切换到色彩转轮视图。转轮视图如下图所示。
色彩转轮上有五个环形色彩标签，分别对应于结果窗口中的五个色彩方块（正中间方块的颜色为主色彩）。您可以通过拖动色彩标签来改变对应的颜色，也可以通过调节结果窗口里的 R、G、B 或者 H、S、V 滑动条等方式来精确设定其颜色。与此同时, 色彩转轮上其他标签的位置和颜色也将根据所选的色彩调和规则发生相应的变化。调和规则可以在规则窗口中设置。

规则窗口

在规则窗口中，您可以设置色彩组的调和和同步规则。

调和规则

调和规则规定了色彩的搭配方式。

  • 相似组
    相似组使用色彩转轮上等夹角且相邻的一组色彩。相似组的颜色之间通常能够很好地相互融合，达到和谐悦目的效果。相似组色彩如下图所示。

使用相似组色彩着色的图片示例。

  • 单色组
    单色组使用色彩转轮上色调（H）相同但饱和度（S）和亮度（V）不同的一组色彩。单色组的颜色搭配在一起通常能够产生舒缓的效果。单色组色彩如下图所示。

使用单色组色彩着色的图片示例。

  • 三色组
    三色组使用色彩转轮上三等分夹角的一组色彩。三色组的颜色搭配在一起能够产生对比效果，但没有补色组的对比那么强烈。三色组色彩如下图所示。

使用三色组色彩着色的图片示例。

  • 四色组
    四色组使用色彩转轮上两两互补的一组色彩。四色组的颜色搭配在一起也能够产生对比效果，形成冷暖对比。四色组色彩如下图所示。

使用四色组色彩着色的图片示例。

  • 五色组
    五色组使用色彩转轮上均匀分布的一组色彩。五色组的颜色搭配在一起能够使画面具有节奏感。五色组色彩如下图所示。

使用五色组色彩着色的图片示例。

  • 补色组
    补色组使用色彩转轮上彼此对立的一组色彩。补色组的颜色对比度非常高，搭配在一起往往很突出，在高饱和度的情况下甚至会引起色彩的不稳定感。补色组色彩如下图所示。

使用补色组色彩着色的图片示例。

  • 浓度组
    浓度组使用色彩转轮上色调（H）和饱和度（S）相同但亮度（V）不同的一组色彩。浓度组的颜色搭配在一起同样能够产生舒缓的效果。浓度组色彩如下图所示。

使用浓度组色彩着色的图片示例。

  • 自定组
    自定组允许您自己手动定义每一个色彩而不受任何规则的约束，从而产生您所期望的效果。自定组色彩如下图所示。

使用自定组色彩着色的图片示例。

同步规则

同步规则规定了色彩标签的移动行为。

  • 不限制
    不限制色彩标签的移动。当您移动主色彩的标签时，其他标签的位置将按照调和规则发生同步移动。

  • 固定 H
    固定所有色彩标签的色调（H）。当您移动主色彩的标签时，只能同步改变所有色彩饱和度（S）和亮度（V），并且其他标签在同步时也会受到调和规则的约束。

  • 固定 S
    固定所有色彩标签的饱和度（S）和亮度（V）。当您移动主色彩的标签时，只能同步改变所有色彩的色调（H），并且其他标签在同步时也会受到调和规则的约束。

  • 等间距
    固定所有色彩标签的间距。当您移动主色彩的标签时，能够同步改变所有色彩的色调（H）、饱和度（S）和亮度（V）而相邻色彩间的夹角保持不变，并且其他标签在同步时也会受到调和规则的约束。

操作窗口

在操作窗口中，您可以执行文档和视图操作。

文档操作

当您搭配好一组或者多组色彩后，您可以将当前结果区域中的色彩组或者仓库中所有的色彩组导出或者保存为可读写文档或者其他格式文档, 以便后续的备份和使用。您也可以在软件中导入或者打开已有的色彩文档。

  • 打开
    在软件中打开色彩仓库文档。支持的文档格式有：DigiPale 色彩仓库文档（*.dpc）和 DigiPale Json 文档（*.json）。
  • 保存
    将色彩仓库保存为本地文档。支持的文档格式有：DigiPale 色彩仓库文档（*.dpc）、纯文本文档（*.txt）、Adobe 样板文档（*.aco）、GIMP 色板文档（*.gpl）和 Pencil 色板文档（*.xml）。
  • 导入
    在软件中导入色彩组文档。支持的文档格式有：DigiPale 色彩组文档（*.dps）和 DigiPale Json 文档（*.json）。
  • 导出
    将色彩组导出为本地文档。支持的文档格式有：DigiPale 色彩组文档（*.dps）、纯文本文档（*.txt）、Adobe 样板文档（*.aco）、GIMP 色板文档（*.gpl）和 Pencil 色板文档（*.xml）。

视图操作

通过内置的视图操作，您可以切换到色彩转轮视图来创建一组色彩，可以切换到色彩图像视图来定位一组色彩，也可以将创建的色彩组添加到色彩仓库中。详情请参见工作区域。

处理窗口

在处理窗口中，您可以对图像进行提取、滤波、快照、裁剪和缩放处理。

提取处理

您可以使用提取处理的功能来自动从图像中定位一组色彩。与此同时，调和规则将自动变为自定组。

  • 明亮多彩
    从图片中提取的色彩组以明亮色彩为主，以浅淡或暗淡色彩为辅，体现颜色的多彩性。

  • 暗淡多彩
    从图片中提取的色彩组以暗淡色彩为主，以浅淡色彩为辅，体现颜色的多彩性。

  • 明亮
    从图片中提取明亮的色彩组，体现颜色的均一性。

  • 浅淡
    从图片中提取浅淡的色彩组，体现颜色的均一性。

滤波处理

您可以使用滤波处理的功能改善图片的质量，以更好地从图片中定位色彩组。注意，滤波处理操作不可逆。

模式窗口

在模式窗口中，您可以设置色彩组结果的显示模式。

显示模式

在显示模式中，您可以显示或者隐藏结果窗口中的 R、G、B 或者 H、S、V 滑动条。

  • RGB
    勾选 RGB 选项框时显示结果窗口中的 R、G、B 滑动条，取消勾选则隐藏滑动条。

  • HSV
    勾选 HSV 选项框时显示结果窗口中的 H、S、V 滑动条，取消勾选则隐藏滑动条。

结果窗口

结果窗口内包含五个色彩方块，方块的颜色与色彩转轮上标签的颜色相对应。双击色彩方块，您可以在弹出的对话框里直接修改颜色。色彩转轮上的标签也会随之发生变化。另外，弹出的对话框里提供了拾取屏幕色彩的功能，可以直接获取当前屏幕下其他窗口的颜色。

五个色彩方块下依次为色彩的六进制字符、RGB 色彩滑动条和 HSV 色彩滑动条，您也可以通过改变这些数值达到直接改变颜色的目的。另外，滑动条的显示与隐藏可以通过模式窗口来控制。

常见问题

如何修改界面语言？

  1. 单击菜单栏中的“编辑”，然后选择“设置”，打开设置对话框。
  2. 点击“软件”选项卡，在“语言”旁的下拉菜单内选择目标语言。

如何导出色彩组文档并导入到其他图像编辑器中？

  1. 在操作窗口文档按钮框内点击“导出”，将这组色彩导出为 Adobe 样板文档（\*.aco）。
  2. Adobe Photoshop：首先打开 Adobe Photoshop，在色板选项卡中下拉菜单中选择“导入”，打开文件对话框。然后找到之前保存的样板文件，点击打开即可。配好的色彩即会追加至当前色板最后。
  3. GIMP：首先打开 GIMP，在色板选项卡中右键选择“导入新的色板”，打开导入对话框。然后在对话框中选择之前保存的样板文件，点击“导入”即可。
  4. 优动漫 Paint（或 Clip Studio Paint）：首先打开优动漫 Paint（或 Clip Studio Paint），在色板选项卡中下拉菜单中选择“载入色板”，打开文件对话框。然后找到之前保存的色板文件，点击打开即可。配好的色彩即会出现在一个单独的色板选项卡中。
